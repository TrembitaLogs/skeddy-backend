# Skeddy Backend

Coordination backend for the Skeddy driver platform — pairs a user-facing mobile app with a device-side search app to discover, filter, and accept ride offers in real time.

## Stack

FastAPI · PostgreSQL · Redis · SQLAlchemy 2.0 (async) · Alembic · Firebase Cloud Messaging · Google Play Billing · Sentry · Docker · GitHub Actions

## Status

Source published for transparency. Closed to external contributions.
